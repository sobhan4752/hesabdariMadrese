# 🏫 سیستم مدیریت شهریه – دبیرستان غیردولتی فرهنگ دوره اول

یک نرم‌افزار تحت وب تک‌کاربره برای مدیریت کامل شهریه و پرداخت‌های دانش‌آموزان.  
مناسب برای مدارسی که نیاز به پیگیری دقیق مالی، ثبت بدهی‌ها، گزارش‌گیری و بک‌آپ خودکار دارند.

## ✨ ویژگی‌های اصلی

- **مدیریت دانش‌آموزان**: بارگذاری از فایل Excel، جستجوی زنده، فیلترهای پیشرفته (کلاس، پایه، وضعیت مالی)
- **ثبت پرداخت و تخفیف**: با قابلیت تخصیص به بدهی‌های سفارشی یا سال‌های قبل
- **کارت حساب جامع**: نمایش شهریه، بدهی‌ها، تخفیف‌ها، پرداخت‌ها و بدهی سال‌های گذشته
- **بدهی‌های سفارشی**: تعریف بدهی‌های دستی (کتاب، فوق‌برنامه و ...) برای هر دانش‌آموز
- **بک‌آپ خودکار**: بک‌آپ کامل پایگاه‌های داده و فایل‌های سیستم با Cron Job
- **تنظیمات خروجی PDF**: شخصی‌سازی کامل ظاهر PDF (فونت، رنگ، چیدمان، واترمارک، امضا)
- **صفحه‌بندی و مرتب‌سازی**: جداول با قابلیت Ajax، شماره ردیف، و کش هوشمند
- **گزارش‌های متنوع**: مقایسه پایه‌ها و کلاس‌ها، تغییر شهریه، سبد حذف‌شده‌ها
- **امنیت**: CSRF Protection، لاگین امن، رمزنگاری نشست، جلوگیری از SQL Injection
- **پنل دیباگ**: بررسی سلامت سیستم، فضای دیسک، یکپارچگی دیتابیس، وضعیت کش و ...
- **رابط کاربری واکنش‌گرا**: طراحی با Tailwind CSS و کاملاً مناسب موبایل و تبلت

## 🚀 نصب و راه‌اندازی

۱. **مخزن را کلون کنید** (یا فایل‌ها را در `public_html` آپلود کنید):
```bash
git clone https://github.com/sobhan4752/hesabdariMadrese.git
cd school-fee-manager
```

۲. **وابستگی‌ها را نصب کنید** (اگر پوشهٔ `vendor` وجود ندارد):
```bash
composer install
```

۳. **پوشهٔ `private` را ایجاد کنید**:
یک پوشه خارج از `public_html` (مثلاً `/home/user/privates`) بسازید و مسیر آن را در فایل `includes/init.php` در ثابت `PRIVATE_PATH` تنظیم کنید.

۴. **مجوزهای لازم را اعمال کنید**:
```bash
chmod -R 755 /home/user/privates
```

۵. **نصب اولیه را اجرا کنید**:
مرورگر را به آدرس `https://your-domain.com/install.php` باز کنید تا پایگاه دادهٔ اصلی (`school_finance.sqlite`) و جداول اولیه ایجاد شود.

۶. **وارد سیستم شوید**:
نام کاربری: `admin`  
رمز عبور: `admin` (پس از اولین ورود، رمز را تغییر دهید)

۷. **بک‌آپ خودکار را تنظیم کنید**:
یک Cron Job با دستور زیر ایجاد کنید (هر شب ساعت ۲ بامداد):
```bash
0 2 * * * wget -q -O /dev/null "https://your-domain.com/backup.php?cron_key=YOUR_SECRET_KEY" >/dev/null 2>&1
```
کلید `YOUR_SECRET_KEY` را از صفحهٔ `backup.php` دریافت کنید.

## 📂 ساختار پروژه

```
.
├── assets/                 # فایل‌های CSS، JS، و کتابخانه‌های استاتیک
├── includes/
│   ├── init.php            # هستهٔ مرکزی، اتصال به دیتابیس، CSRF
│   ├── functions.php       # توابع کمکی (محاسبه مانده، کش، فرمت مبلغ)
│   ├── header.php          # هدر، سایدبار و منو
│   └── footer.php          # فوتر
├── private/ (خارج از ریشه) # پایگاه‌های داده، بک‌آپ، کش، فونت‌ها، لوگوها
│   ├── school_finance.sqlite  # پایگاه دادهٔ اصلی
│   ├── year_*.sqlite          # پایگاه‌های دادهٔ سال‌های مالی
│   ├── backups/               # فایل‌های بک‌آپ ZIP
│   ├── cache/                 # کش ترازنامه‌ها
│   ├── fonts/                 # فونت‌های آپلودشده
│   └── logos/                 # لوگو و مهرهای آپلودشده
├── vendor/                 # کتابخانه‌های Composer
├── index.php               # پیشخوان
├── students.php            # لیست دانش‌آموزان
├── payment.php             # ثبت پرداخت
├── student_card.php        # کارت حساب دانش‌آموز
├── backup.php              # مدیریت بک‌آپ
├── settings.php            # تنظیمات سیستم
├── debug.php               # صفحهٔ دیباگ
├── install.php             # نصب اولیه (پس از نصب حذف شود)
└── README.md
```

## ⚠️ نکات امنیتی

- فایل‌های `private/` باید **کاملاً خارج از دسترس وب** باشند (از طریق `.htaccess` یا مسیر خارج از `public_html`).
- پس از نصب، فایل `install.php` را **حذف** کنید.
- رمز عبور پیش‌فرض را بلافاصله تغییر دهید.
- از پروتکل HTTPS استفاده کنید.
